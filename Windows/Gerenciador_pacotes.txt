**Comandos
PARA VERIFICAR O NOME DOS PACOTES, ACESSAR https://chocolatey.org/

INSTALANDO PACOTES
choco install <nome-do-pacote> = instala um pacote/programa
choco install -y

REMOVENDO PACOTES
choco uninstall <nome_do_pacote> = remove o pacote 

PACOTES LOCAIS
choco list
choco list -l
choco search
